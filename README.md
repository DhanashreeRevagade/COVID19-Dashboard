# COVID19-Dashboard

A live updated COVID-19 Dashboard to visualize the deaths, confirmed and recovered cases all around the world. 
!(Imgs/Homepage.png)

Tools/Libraries
--
Python 3, Streamlit package, libraries - plotly, numpy, pandas
